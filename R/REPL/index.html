발음

설명

대화식 최상위 레벨 또는 언어 쉘이라고도하는 REPL (읽기-판정 프린트 루프)은 단일 사용자 입력을 가져 와서 실행하고 결과를 사용자에게 반환하는 간단한 대화형 컴퓨터 프로그래밍 환경입니다. REPL 환경에서 작성된 프로그램은 piecewise 단위로 실행됩니다. 이 용어는 일반적으로 클래식 Lisp 머신 대화식 환경과 유사한 프로그래밍 인터페이스를 나타내는 데 사용됩니다. 일반적인 예로는 명령 줄 셸 프로그래밍 언어와 유사한 환경이 있으며 또한 이 기술은 스크립팅 언어의 특징입니다.

python 혹은 node를 터미널에서 실행할 수 있는 기술이다.
